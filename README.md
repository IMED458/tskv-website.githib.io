# Tsivis Kveli Website

მრავალგვერდიანი ვებ-საიტი `tsiviskveli.ge`-ზე არსებული ინფორმაციის საფუძველზე.

## რა არის გადმოტანილი
- სრული პროდუქტების კატალოგი (`20/20` პოზიცია) სრული ტექსტური აღწერებით
- „ჩვენს შესახებ“, „გამოცდილება“, „მომსახურეობა“, „ხარისხი“, „გიდებისთვის“, „მაღაზიებისთვის“, „ტურისტებისთვის“, „კონტაქტი“ ტექსტები
- ფოტომასალა და გალერეა
- Facebook და Instagram აიქონები/ბმულები header/footer-ში

## გვერდები
- `index.html` - მთავარი
- `about.html` - ჩვენს შესახებ
- `products.html` - პროდუქტები (სრული ჩამონათვალი)
- `services.html` - სერვისები (სრული ტექსტები)
- `gallery.html` - გალერეა + ბოლო სიახლეები
- `contact.html` - კონტაქტი

## ფაილები
- `assets/css/styles.css` - საერთო დიზაინი
- `assets/js/main.js` - მობილური მენიუ + მონაცემების რენდერი
- `assets/data/content.json` - ტექსტური ბლოკები
- `assets/data/products.json` - სრული პროდუქციის სია
- `assets/data/news.json` - სიახლეები
- `assets/images/` - ფოტოები
- `source-scrape/` - წყაროდან ჩამოწერილი HTML რეფერენსისთვის

## გაშვება
სტატიკური ფაილებია. გახსენი `index.html` ბრაუზერში ან გამოიყენე ნებისმიერი static server.

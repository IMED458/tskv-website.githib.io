# Tsivis Kveli Website

მრავალგვერდიანი (multi-page) ვებ-საიტი `tsiviskveli.ge`-ზე არსებული კონტენტის საფუძველზე.

## გვერდები
- `index.html` - მთავარი
- `about.html` - ჩვენს შესახებ
- `products.html` - პროდუქტები
- `services.html` - სერვისები
- `gallery.html` - გალერეა
- `contact.html` - კონტაქტი

## სტრუქტურა
- `assets/css/styles.css` - საერთო სტილები
- `assets/js/main.js` - მობილური მენიუ
- `assets/images/` - საიტის ფოტოები
- `source-scrape/` - წყაროდან ჩამოწერილი HTML ფაილები (რეფერენსისთვის)

## გაშვება
სტატიკური ფაილებია; გახსენი `index.html` ბრაუზერში ან გაუშვი ნებისმიერი static server.
